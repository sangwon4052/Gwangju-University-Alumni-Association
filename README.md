# 🎓 광주대학교 총동문회 관리시스템 (개선 버전)

> **Gwangju University Alumni Association Management System**  
> GitHub Pages로 즉시 배포 가능한 순수 HTML 웹 애플리케이션

---

## 🆕 새로운 기능 (2024 업데이트)

### 1. 💰 동문기부 시스템
- **홈 화면에 기부금 총액 표시**: 누적 기부금을 실시간으로 확인 가능
- **기부 신청 양식**: 
  - 학과, 동문명, 기부금 입력
  - 영수증 종류 선택 (현금영수증 / 세금계산서)
  - **현금영수증**: 휴대폰 번호 입력
  - **세금계산서**: 사업자등록번호, 이메일 입력
- **관리자 기능**:
  - 전체 기부 내역 조회
  - 엑셀 다운로드
  - 개별 기부 삭제
- **공개 목록**: 최근 기부 내역 10건 표시

### 2. 📄 회의록/업무추진비 파일 열람
- **회원 접근 가능**: 모든 회원이 회의록과 업무추진비 파일을 다운로드하거나 볼 수 있습니다
- **파일 관리 개선**:
  - 이미지 파일: "보기" 버튼으로 바로 확인
  - 문서 파일: "다운로드" 버튼으로 다운로드
  - 관리자는 파일 업로드 및 삭제 가능

### 3. 🏠 홈 화면 개선
- 가입 동문 수와 함께 **누적 기부금** 실시간 표시
- 시각적으로 개선된 배지 디자인

## 🌐 바로 사용하기 (GitHub Pages 배포)

### 1단계 — GitHub 저장소 만들기
1. [github.com](https://github.com) 접속 → 로그인
2. 우측 상단 `+` 버튼 → **New repository** 클릭
3. Repository name: `gwangju-alumni` (원하는 이름 가능)
4. **Public** 선택 (Pages 무료 배포용)
5. **Create repository** 클릭

### 2단계 — 파일 업로드
1. 저장소 페이지에서 **Add file → Upload files** 클릭
2. `index.html` 파일을 드래그하거나 선택
3. 아래 Commit changes 버튼 클릭

### 3단계 — GitHub Pages 활성화
1. 저장소 상단 **Settings** 탭 클릭
2. 좌측 메뉴에서 **Pages** 클릭
3. Source: **Deploy from a branch**
4. Branch: **main** / **/ (root)** 선택 후 **Save**
5. 잠시 후 상단에 주소 표시:  
   👉 `https://[내GitHub아이디].github.io/gwangju-alumni/`

> ⏱️ 배포까지 약 **1~3분** 소요됩니다.

## 📁 파일 구조

```
gwangju-alumni/
├── index.html      ← 메인 파일 (이 파일 하나로 전체 동작)
└── README.md       ← 이 설명 파일
```

> **단 1개의 파일(index.html)** 으로 모든 기능이 동작합니다.  
> 이미지, 스타일, 스크립트가 모두 포함되어 있습니다.

| 메뉴 | 기능 |
|------|------|
| 🏠 **홈** | 가입 동문 수 및 **기부금 총액** 실시간 표시, 주요 행사 미리보기, 공지사항 |

## 💾 데이터 저장 방식

- 모든 데이터는 **브라우저 localStorage**에 저장됩니다
- 같은 기기 · 같은 브라우저에서 데이터가 유지됩니다
- 브라우저 캐시 삭제 시 데이터가 초기화될 수 있습니다

> 📌 **중요**: 회원 명단, 기부 내역 등 중요 데이터는 주기적으로 **엑셀 다운로드**하여 백업하시길 권장합니다.
