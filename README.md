# 광주대학교 총동문회 웹사이트 - 완전 재구축 버전

## 📋 개요
광주대학교 총동문회 웹사이트에 요청하신 세 가지 주요 기능이 모두 구현되었습니다.

## ✅ 구현된 기능

### 1. 회의록·업무추진비 파일 다운로드 기능
- **모든 회원이 파일을 다운로드**하여 열람 가능
- 이미지 파일: "보기" 버튼으로 새 창에서 열기
- 기타 파일: "다운로드" 버튼으로 다운로드
- 관리자는 파일 업로드 및 삭제 가능

### 2. 동문기부 메뉴 추가
- **네비게이션 메뉴**: PC 및 모바일 메뉴에 "동문기부" 추가
- **입력 폼**:
  - 학과 (필수)
  - 동문명 (필수)
  - 기부금 (원, 필수)
  - 영수증 종류 (필수): 현금영수증 / 세금계산서
    - **현금영수증 선택 시**: 휴대폰 번호 입력
    - **세금계산서 선택 시**: 사업자 번호, 이메일 입력
- **기부 내역 목록**: 
  - 일반 회원: 최근 10건만 표시
  - 관리자: 전체 내역 표시 + Excel 다운로드 기능

### 3. 홈 화면 기부금 총액 표시
- "가입 동문" 배지 옆에 **"총 기부금"** 배지 추가
- 실시간으로 총 기부금액 합계 표시
- 자동 업데이트 (페이지 로드 시, 기부 등록 시)
